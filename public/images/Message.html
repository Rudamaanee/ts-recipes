첨부파일 다운로드
열람 안내 * 삼성계열 기본 브라우저와 네이버 앱(안드로이드기종)은 상단 첨부파일 다운로드가 지원되지 않습니다.

* Chrome, FireFox 브라우저 등을 이용하여 보안메일 html 파일을 다운로드 후, 다운로드된 보안메일 html 파일을 VestMail 전용뷰어 로 열람하여 주십시요.

> VestMail 전용뷰어를 통한 열람시 아래와 같은 과정으로 열람이 가능합니다.

  1) VestMail 설치.
  2) 메일 열람.
  3) 보안메일 html 파일 다운로드.
  4) 다운로드된 보안메일 html 파일 선택.
  5) 열람 가능앱 리스트에서 "VestMail" 선택.
  6) 비밀번호 입력 후 첨부파일 선택하여 파일 저장(열람).

열람 안내 * Chrome, Firefox, Dolphin 브라우저는 상단 첨부파일 다운로드가 지원되지 않습니다. Safari 또는 Opera 브라우저를 이용하여 주십시요.

* 다음 앱 이용시 "웹으로 미리보기"는 상단 첨부파일 다운로드가 지원되지 않습니다. "임시저장 후 미리보기" 선택후 "VestMail" 로 열람하여 주십시요.

> VestMail 전용뷰어를 통한 열람시 아래와 같은 과정으로 열람이 가능합니다.

  1) VestMail 설치.
  2) 메일 열람.
  3) 보안메일 html 파일 선택.
  4) "보안메일의 복호화를 진행할 수 없습니다." 라는 화면에서 iOS 공유 아이콘 선택.
  5) 공유 앱 리스트에서 "VestMail에 복사" 선택.
  6) 비밀번호 입력 후 첨부파일 선택하여 파일 저장(열람).

  * Safari, Opera 이용시 네이버, 다음 웹메일은 "VestMail" 설치가 불필요 합니다.
